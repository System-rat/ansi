Detailed Description

Holds color information compatible with ANSI standards.

Holds various formats of colors for use with other functions/classes to represent ANSI compatible color formats

Member Enumeration Documentation

◆ ColorType

enum ansi::StyleColor::ColorType

The format type of the color.

Term256 is for chosing one of the 256 builtin colors, RGB is for chosing the Red, Green, and Blue channels individually (on supported terminals), and ANSI is for the 8 ANSI colors
